Contribution

Data Visualization Lab では ArcGIS for Developers を使って作成したビジュアライゼーションをこのサイトに追加できます。追加方法は本サイトを管理している GitHub リポジトリの CONTRIBUTION.md をご参照ください。

Data Visualizaion Lab とは？

ArcGIS のオープンなプラットフォームを基盤として、さまざまなオープン テクノロジーを組み合わせたデータビジュアライゼーションの可能性を探求しています。

本サイトで紹介しているビジュアライゼーションは非商用目的であればすべて無償で公開が可能なものです。データの配信や、マップの作成、アプリ開発のための API 利用には ArcGIS for Developers アカウントが必要です。
すぐに始めてみたい方向けの開発リソース集を公開していますので併せてご覧ください。
